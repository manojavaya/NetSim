# MoSCoW Prioritization – NetSim

## Must Have
- Create and visualize network topologies
- Configure network nodes
- Test communication between nodes
- Execute distributed tasks
- Role-based access control
- Secure data transmission
- Local deployment support

## Should Have
- Real-time network activity monitoring
- Performance benchmarking of tasks
- Resource usage monitoring
- System logging and notifications
- Save and load simulation configurations

## Could Have
- File sharing between nodes
- In-system messaging
- Advanced visualization enhancements
- Extended protocol customization

## Won’t Have (For Now)
- Large-scale cloud deployment
- Real-world hardware integration
- Automated AI-based optimization
